# ASCII Game Hub by ASC11AM0N

A terminal-based game launcher that detects and displays ONLY installed games on your system. Features themed ASCII art and a clean, categorized interface.

## Features

- Themed ASCII art headers for different game categories
- Automatically detects only installed games
- Categories: Story/Roguelike, Action/Arcade, Boards/Classics
- Clean terminal UI with centered layouts
- One-click game launching

## Categories

### Story / Roguelike
- Rogue, Nethack, Angband, Crawl, and more!

### Action / Arcade
- Robotfindskitten, Tetris, Snake, Moon-buggy, etc.

### Boards / Classics
- Chess, Sudoku, Netris, 2048, and classic board games
